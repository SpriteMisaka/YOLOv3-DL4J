# YOLOv3-DL4J
A JAVA application using DL4J to load keras YOLOv3 model to detect objects on VisDrone dataset.

Note:
+ Make sure you have your own YOLOv3 keras model. You can train your keras YOLOv3 model using [https://github.com/susnato/keras-yolo3](https://github.com/susnato/keras-yolo3) or [https://github.com/qqwweee/keras-yolo3](https://github.com/qqwweee/keras-yolo3).
+ Modify the classes file in the resources folder if you do not use VisDrone2019 dataset.
